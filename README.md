# Hyperf-core component

Includes the following general usage classes:

- Command:
    - AnonymizeDataCommand.
- DTO:
    - AbstractDTO;
    - PaginationRequestDTO;
    - PaginationResultDTO.
- Model:
    - AbstractModel;
    - AbstractFilter.
- Repository:
    - AbstractRepository.
- Service:
    - AnonymizeDataService;
    - Service #[Attribute].

Install:
```
composer require hyperf-core/core
```
